University Youth Festival Website


The project entitled “University Fest ” is a website, which provides students to register for university kalolsav through online. Students can visit the website from the comfort of their homes and register for the program items they wish to participate by signing in with their college name and password provided by the university at their home. The students can search for the details of the scheduling of the events and they can know the latest informations about the kalolsav including the result. For this software ASP.NET is used as a front end and SQL SERVER is used as backend.
